# Raz

Raz is a personal launcher and embedded homepage. Launch local apps and games, monitor server and device status, and manage bookmarks — all without opening an external browser.

## Key Features

* **Local Launcher** — Launch games and applications directly from Raz
    * Configurable launch options: paths, environment (e.g. `.venv`), arguments, and more
    * Right-click context menu to configure apps, open their directories, etc.
* **Embedded Homepage** — Multi-tab page with live editing (add/remove content, drag-and-drop reordering)
    * Bookmarked links with preview thumbnails
    * Live status indicators for servers, devices, and webpages
